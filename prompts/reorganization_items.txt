Every item should have these:

```ts
Item {
  id: string
  name: string
  type: "weapon" | "armor" | "tool" | "consumable" | "gear" | "magic" | "treasure"
  description: string

  weight: number | null
  cost: {
    amount: number
    currency: "cp" | "sp" | "ep" | "gp" | "pp"
  } | null

  source: string        // PHB, DMG, Homebrew
  tags: string[]        // "light", "martial", "stealth", etc.
}
```

üí° `tags` are powerful ‚Äî they let users filter without new fields.
