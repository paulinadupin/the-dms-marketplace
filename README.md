# The DM's Marketplace

A web application for Dungeon Masters to create virtual marketplaces for their D&D campaigns.

---

## What is This?

The DM's Marketplace helps Dungeon Masters create interactive shopping experiences for their D&D games. DMs can build item libraries, create shops, and manage virtual marketplaces. Players can browse items, make purchases with virtual currency, and track their inventory.

---

## Features

**For Dungeon Masters:**
- Create and manage item libraries (custom or D&D 5e official items)
- Build shops with custom pricing and inventory
- Create markets with shareable access codes for players
- Track player purchases and inventory in real-time

**For Players:**
- Browse shops using an access code from your DM
- Purchase items with virtual gold, silver, and copper
- Manage your character's inventory
